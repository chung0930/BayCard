# BayCard 快速開始指南

## 🎯 5 分鐘快速上手

### 1. 訪問應用

開發環境已在運行，訪問：
```
https://3000-imhxfrilhvwo6zb3498fb-3d7e5e82.sg1.manus.computer
```

### 2. 登入帳戶

點擊右上角 **Sign In** 按鈕，使用 Manus OAuth 登入。

### 3. 發布第一篇文章

1. 點擊 **New Article** 按鈕
2. 填寫以下信息：
   - **Title**: 文章標題
   - **Board**: 選擇看板（例如：感情、時事、校園）
   - **Content**: 文章內容
3. 點擊 **Publish Article**

### 4. 查看文章

- 首頁顯示熱門文章列表
- 點擊任何文章進入詳情頁
- 在詳情頁可以：
  - ❤️ 按讚
  - 🔖 收藏
  - 💬 留言

### 5. 管理個人資料

1. 點擊右上角用戶名
2. 選擇 **Profile**
3. 點擊 **Edit Profile** 編輯信息

## 📱 主要頁面

| 頁面 | URL | 功能 |
|------|-----|------|
| 首頁 | `/` | 瀏覽熱門文章、看板列表 |
| 文章詳情 | `/article/:slug` | 查看文章、留言、按讚 |
| 個人頁面 | `/profile` | 查看個人文章、編輯資料 |
| 發布文章 | `/create-article` | 發布新文章 |

## 🎨 界面特點

- **簡潔白底設計** - 優雅的 Dcard 風格
- **響應式佈局** - 完美支持桌面和手機
- **暗色主題** - 支持亮色/暗色切換（可在設置中調整）
- **流暢動畫** - 優化的用戶體驗

## 🔍 常見操作

### 搜尋文章
目前可通過看板篩選文章，完整搜尋功能即將推出。

### 查看熱門文章
首頁默認顯示按讚數和留言數排序的熱門文章。

### 管理收藏
在文章詳情頁點擊 🔖 圖標收藏文章。

### 查看留言
在文章詳情頁下方查看所有留言。

## ⚙️ 系統要求

- 現代瀏覽器（Chrome、Firefox、Safari、Edge）
- JavaScript 已啟用
- Cookie 已啟用（用於會話管理）

## 🚀 部署到生產環境

### 使用 Manus 平台

1. 在管理界面點擊 **Publish** 按鈕
2. 系統將自動構建和部署
3. 應用將在 `https://baycard.manus.space` 上線

### 配置自定義域名

1. 進入 **Settings > Domains**
2. 購買或綁定自定義域名
3. 配置 DNS 記錄

## 📊 監控和日誌

系統自動記錄：
- 文章瀏覽數
- 用戶操作日誌
- 系統性能指標

在管理界面的 **Dashboard** 查看實時統計。

## 🆘 故障排除

### 無法登入
- 確保 Cookie 已啟用
- 清除瀏覽器緩存
- 嘗試使用隱私窗口

### 文章發布失敗
- 確保標題和內容不為空
- 確保已選擇看板
- 檢查網絡連接

### 頁面加載緩慢
- 刷新頁面
- 清除瀏覽器緩存
- 檢查網絡速度
